Uncertainties

For more extensive propocalculations without units, we use Uncertainties, a  free, cross-platform Python module that  handles calculations with pure numbers with uncertainties (like 9.80Â±0.03). This is in fact what Pint uses under the hood. The uncertainties package returns its result with an uncertainty as predicted by linear error propagation theory, while also correctly taking into account correlations between variables.
