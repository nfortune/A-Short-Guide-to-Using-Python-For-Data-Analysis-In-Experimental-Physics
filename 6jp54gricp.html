Uncertainties

For propagation of uncertainty calculations without units  for variables and arrays , we use the  Uncertainties module written by The uncertainties module returns its result with the uncertainty specified by linear error propagation theory, correctly taking into account any direct correlations between variables.  This is in fact what Pint uses under the hood.
Lebigot recommends two related uncertainty calculation Python packages if you need still more advanced approaches to propagation of uncertaintyy but we have not tried them ourselves yet : soerp (second-order approximations) and mcerp (Monte-Carlo approach).
