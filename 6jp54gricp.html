Uncertainties

For propagation of uncertainty calculations without units , we use  Uncertainties: a Python package written by Eric. O. Lebigot.  The uncertainties module returns its result with the uncertainty specified by linear error propagation theory, correctly taking into account any direct correlations between variables.  This is in fact what Pint uses under the hood. If you need still more advanced approaches to propagation of uncertainty, Lebigot recommends trying  soerp (second-order approximations) and mcerp (Monte-Carlo approach).
