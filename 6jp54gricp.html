Uncertainties

For propagation of uncertainty calculations without units , we use  Uncertainties: a Python package written by Eric. O. Lebigot. Although sufficient for amlosLebigot recommends two related uncertainty calculation Python packages if you need still more advanced approaches to propagation of uncertaintyy but we have not tried them ourselves yet : soerp (second-order approximations) and mcerp (Monte-Carlo approach).
