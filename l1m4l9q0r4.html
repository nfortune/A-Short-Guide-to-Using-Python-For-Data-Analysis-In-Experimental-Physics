Working with measurement data

When using Python in place of a calculator, we have the ability to directly include information about dimensions, units and uncertainties in the calculations. When used, these abilities  offer the advantage of allowing us to check algebraic calculations, automatically propagate calculated uncertaintiesand avoid unit conversion errors, but the necessary Python modules are not included in the standard Anaconda Navigator Python installation and must be added by hand (unless already installed as part of a specially configured webserver).
