Installing Python Packages

This section is only relevant if you are running Python on your own computer. If you are running Python within a Jupyter notebook on a webserver or a computer account which has already been configured for your use (such as https://jove.smith.edu for Smith College physics) , this section can be skipped. 

Installing Anaconda Navigator

Because our focus in this article  is limited to  the use of Python to expedite the  analysis of your experimental data (and not, for example, the specifics of various Python distributions and their relative merits for computational physics in terms of speed, accuracy, and memory requirements)  we limit our discussion here to just one way to install Python, interactive Python  (iPython) and Jupyter notebooks,  scientific python development environments (editing, testing, debugging)  such as Spyder,  and  essential Scientific Python packages such as numpy, matplotlib and scipy on your computer: a free and comprehensive Python distribution and installer package  called Anaconda Navigator.  This is freely available for Mac OS, Linux, and Windows. This is the same package provided on Smith College physics department computers and Jupyter webservers. After downloading and installing  Anaconda Navigator, you are ready to go
We recommend it primarily for its ease of installation, its ease of use,  its very comprehensive set of preinstalled and easily added packages, and its ability to create more than one Python "environment" (each running its own set of packages and/or versions of Python).
