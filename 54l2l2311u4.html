Installing Python Packages

This section is only relevant if you are running Python on your own computer. If you are running Python within a Jupyter notebook on a webserver or a computer account which has already been configured for your use (such as https://jove.smith.edu for Smith College physics) , this section can be skipped. 

Installing Anaconda Navigator

Because our focus in this article  is limited to  the use of Python to expedite the  analysis of your experimental data (and not, for example, the specifics of various Python distributions and their relative merits for computational physics in terms of speed, accuracy, and memory requirements)  we limit our discussion here to just one way of many to install Python, interactive Python (iPy and additional essential Scientific Python packages such as numpy, matplotlib and scipy on your computer: the free distribution and installer provided by Anaconda called Anaconda Navigator.
