The numerical derivative of the filtered signal \(y\left(x\right)\) can then be evaluated at each \(x\) value using the numpy function gradient. For a 1D array of data the gradient will be the same as the derivative \(\frac{df}{dx}\). According to the  numpy.gradient  reference page , this function "calculates the gradient using second order accurate central differences in the interior points and either first or second order accurate one-sides (forward or backwards) differences at the boundaries. "
