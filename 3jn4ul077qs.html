The Authorea server will work for all of the examples except the (aptly named) Python packages Pint and Uncertainties for numerical calculations using units and/or uncertainties, as those packages are not currently available on the Authorea server (but maybe someday?) but is not designed for hosting your own data and programs (unless you are writing an article about them?) nor is it particularly speedy.  If you don't have access to a local Jupyter webserver with the packages you need (or you prefer a different Python interface such as Spyder) , you will want to install and run Python and Jupyter on your own computer.
