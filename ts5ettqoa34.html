For general propagation of uncertainty tasks , we use  Uncertainties: a Python package written by Eric. O. Lebigot.  The uncertainties module returns its result with the uncertainty specified by linear error propagation theory, correctly taking into account any direct correlations between variables.  Pint uses Uncertainties (under the hood)!
