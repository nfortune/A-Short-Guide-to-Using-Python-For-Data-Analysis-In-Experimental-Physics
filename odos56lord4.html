This method is used,  for example,  in the Python-based textbook Computational Physics by Mark Newman.  This is, arguably, the simplest, easiest to understand method. 
Note for Python geeks: If you wanted to import all modules from numpy at once, you coul type from numpy import * instead of from numpy import loadtxt. Although convenient, it does have some drawbacks: (1) you are importing code you don't need (2) that additional code might include functions with the same name as one you are already using (which would overwrite your earlier definition). We don't recommend it unless you are using a lot of different numpy modules all at once.
The second method imports all numpy modules at once. We call this the traditional numpy import style.
