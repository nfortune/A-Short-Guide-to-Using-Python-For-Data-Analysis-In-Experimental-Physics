This method is used,  for example,  in the highly recommended Python-based textbook Computational Physics by Mark Newman \cite{mark2013}.  To instead import all numpy modules at once using this method, type from numpy import *. The import * command is  dangerously  convenient — we will sometimes use it ourselves, being not as pure in thought as others might be on this issue— but take two considerations in mind: (1) you are importing code you don't need (2) you will overwrite any already imported  functions with the same name as a numpy function with the newer numpy version. 
The second method requires a prefix before each use of a module. We call this the traditional numpy import style.
