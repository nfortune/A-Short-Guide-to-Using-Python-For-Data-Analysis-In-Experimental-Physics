Note:  ordinarily, savetxt would save the data as three rows of data with each variable corresponding to a different row instead of to a different column.  That is often convenient for work within python but is not what we wanted  in this particular case.  We therefore did the following trick before saving the data to a file: we used the 2D array transpose command  to create a 2D matrix of our data, then transposed the matrix to flip the rows and columns.
