For first-order ('linear') error propagation, you can either Pint's plus_minus() operator or the Uncertainties package by Eric O. Lebigot (used by Pint). Uncertainties is a free, cross-platform program that  handles calculations with numbers with uncertainties (like 9.80Â±0.03). The package returns its result with an uncertainty as predicted by linear error propagation theory. As part of that calculation, it also automatically calculates derivatives.  Correlations between variables are also automatically handled.
