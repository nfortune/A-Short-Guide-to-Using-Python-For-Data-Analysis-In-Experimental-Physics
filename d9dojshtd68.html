loading data from a CSV file 

Our task is to use the Numerical Python (numpy) command loadtxt to load data from a .csv text file with two columns of data (say x and y ) and one row of text preceding the data.  Each column of data will become a Numerical Python (numpy) array.  The csv file is the Calibration_650nm.csv file shown earlier in section \ref{313963}  with array names  angle  V_pd, and delta_V_pd .  Note that we use the CSV file format here because it is ubiquitous and easy to export/import using spreadsheets. If your data is in a text file but uses a different file format (such as tab-delimited) you can still use  loadtxt to import your data if you modify the delimiter = ',' command used in the examples shown here  to suit your .  See the numpy.loadtxt manual page for details. 
loadtxt is not part of the core Python language but is instead part of a package we must "import" into our program before using. There are two common methods of doing this.
In the first method, we import each individual function  or module such as loadtxt from numpy as needed . We call this the direct numpy import style.
