loading data from a CSV file 

Our task is to use the Numerical Pnumpy command loadtxt to load data from a .csv text file with two columns of data (say x and y ) and one row of text preceding the data.  Each column of data will become a Numerical Python (numpy) array.  We use the CSV file format here because it is ubiquitous and easy to export/import using spreadsheets. If your data is in a text file but uses a different file format (such as tab-delimited) you can still use  loadtxt to import your data. See the numpy.loadtxt manual page for details.
