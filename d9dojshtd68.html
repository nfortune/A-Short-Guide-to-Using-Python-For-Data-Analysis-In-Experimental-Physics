loading data from a file

Below are two ways of accomplishing the same task: loading data from a .csv text file with three columns of data (say x, y, and \(\delta y\) ) and one row of text preceding the data.  Each column of data becomes a Numerical Python array. In this example, the csv file is the 650 nm calibration with error.csv file shown earlier in section \ref{313963}  and the array names are angle, V_pd, and V_pd_delta.  In both cases, we will use the loadtxt command from Numerical Python (numpy) to load the data in this .csv file. See the numpy.loadtxt manual page for details. 
This first method imports  individual functions  and modules from numpy as needed. We call this the direct numpy import style.
