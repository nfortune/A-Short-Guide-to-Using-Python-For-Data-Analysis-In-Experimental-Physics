Installing packages not included in Anaconda Navigator default installer

Example: pip 

When needed, it is still  relatively easy to use Anainstall modules  not included in the standard  Anaconda Navigator installer package. from within  Anaconda Navigator using the terminal window.  This screenshot shows how to open a terminal window from with Anaconda Navigator. execute the  'pip install --upgrade pint' command from the terminal window. See documentation page for Pint and screenshot below for an example.  An example of how to start up and use Pint is attached as code to the screenshot. You can view it by clicking on PintTest.ipynb . Unlike most of the examples in this article, however, it won't successfully run (at present) because Pint is not included on the standard install of Authorea's python webserver.  This will be addressed in a future version of the Authorea editor.
