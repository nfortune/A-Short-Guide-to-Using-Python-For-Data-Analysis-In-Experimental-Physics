Importing and exporting data

There are many ways to import, export, and represent data in files. Here we provide just enough to get you started but it might be all you need. In these examples, we assume you first enter the data into a spreadsheet program (such as Microsoft Excel), then export that data in 'CSV' (comma separated variable) file format. This is a plain text file with commas marking where one value ends and another begins. We use the CSV file format here because it is ubiquitous and easy to export/import using spreadsheets. Data in other plain text file formats See the numpy.loadtxt manual page for details.
