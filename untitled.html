Installing packages not included in Anaconda Navigator default installer

Calculations with units

In the  PHY350 Experimental Physics course here at Smith College, we make extensive use of the Python package Pint by Hernan Greco. Although not included in the standard installer package, it is still relatively easy to install using Anaconda Navigator using the 'pip install --upgrade pint' command from the terminal window.
