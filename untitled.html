This is a short guide to using Python as an advanced scientific calculator for data import, plotting, curve fitting, and simple algebraic calculations (including units and uncertainties) , as used in our advanced lab course PHY350 Experimental Physics.  It is not intended as a complete  introduction to scientific programming in Python but instead as a quick start guide to accomplishing some commonly needed tasks when working with data acquired in lab. 

Importing and exporting data

There are many ways to import, export, and represent data in files. Here we provide just enough to get you started but it might be all you need. In these examples, we assume you first enter the data into a spreadsheet program (such as Microsoft Excel), then export that data in 'CSV' (comma separated variable) file format. This is a plain text file with commas marking where one value ends and another begins.

CSV spreadsheet file format

For specificity, let's assume that the data values were entered in a spreadsheet consisting of one or more columns — one for each measured variable — and a few initial 'header' rows of text providing information about the data that we need to skip over when loading numerical values. Here's what the first few lines of a typical spreadsheet might look like if opened as a text file
