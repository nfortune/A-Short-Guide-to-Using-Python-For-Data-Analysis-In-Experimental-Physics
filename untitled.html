This is a short guide to using Python as an advanced scientific calculator for data import, plotting, curve fitting, and simple algebraic calculations (including units and uncertainties) , as  needed in our advanced lab course PHY350 Experimental Physics.  It is not intended as a complete  introduction to scientific programming in Python but instead as a quick start guide to accomplishing commonly needed tasks when working with data. Please take it as a slowly evolving work in progress: comments and suggestions are welcome but may take time to be incorporated!
