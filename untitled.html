This is a short guide to using Python to accomplish some commonly needed tasks when working with data acquired in lab, including data import, plotting, curve fitting, and algebraic calculations (including units and uncertainties) . Many of the examples involve student data collected in our advanced lab course PHY350 Experimental Physics.

Importing and exporting data

There are many ways to import, export, and represent data in files. Here we provide just enough to get you started but it might be all you need. In these examples, we assume you first enter the data into a spreadsheet program (such as Microsoft Excel), then export that data in 'CSV' (comma separated variable) file format. This is a plain text file with commas marking where one value ends and another begins.
