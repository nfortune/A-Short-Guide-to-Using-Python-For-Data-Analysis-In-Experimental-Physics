This is a short guide to using Python to accomplish some commonly needed tasks when working with data acquired in an experimental physics lab, including data import/export, plotting with error bars, curve fitting of functions to data, and useful algebraic calculations involving units and uncertainties . Many of the examples make use of student data collected in our advanced lab course PHY350 Experimental Physics. 
The goal of this guide is that by following the examples and modifying the attached sample Python code, you should be able to get up and running without first knowing how to accomplish advanced programming tasks (or even knowing how to write a for loop). To go further, however, you will need to learn some basic Python syntax (so as to be able to define your own mathematical functions for curve fitting, for example). 
To get started, I recommend reading the very well written chapter  Python programming for physicists from the Python-based textbook Computational Physics, which gives "an introduction to the Python language at a level suitable for readers with no previous programming experience" \cite{mark2013}.  Instead of the IDLE command line editor discussed there, however, we will run Python within the more user friendly Mathematica-style  Jupyter notebook interface. If this is new to you, see the short (and helpful)  online interactive  tutorial on how to run Python inside Jupyter notebooks.
