Using other installers within Anaconda Navigator

When needed, it is still  relatively easy to use Anaconda Navigator to install packages not included in the standard  Anaconda Navigator installer package. 
  1. open a terminal window from with Anaconda Navigator (see below)
  2. issue the installation command specified in the package documentation (usually conda or pip)
  3. when using Anaconda, the conda installer is preferable to the pip installer , if available (as it will usually also install any additional needed extensions)
