Using other installers within Anaconda Navigator

When needed, it is still  relatively easy to use Anaconda Navigator to install packages not included in the standard  Anaconda Navigator installer package.  The first step is to open a terminal window from with Anaconda Navigator. The second step is to issue the installation command.
