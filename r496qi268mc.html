Using other installers within Anaconda Navigator

When needed, it is still  relatively easy to use Anaconda Navigator to install packages not included in the standard  Anaconda Navigator installer package. 
  1. open a terminal window from with Anaconda Navigator (see bel
  2. The second step is to issue the installation command.
  3. If using Anaconda, the conda installer is preferable to the pip installer (as it will usually also install any additional needed extensions)
