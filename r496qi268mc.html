Using other installers

When needed, it is still  relatively easy to use Anaconda Navigator to install packages not included in the standard  Anaconda Navigator installer.
  1. open a terminal window from with Anaconda Navigator (see figure \ref{231311} below )
  2. issue the installation command. See below for examples using conda  (section \ref{240339}) and pint (section \ref{262745})
Note: When using Anaconda, the conda installer is preferable, if available (as conda will usually also install any additional needed extensions)
