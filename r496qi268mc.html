Using other installers within Anaconda Navigator

When needed, it is still  relatively easy to use Anaconda Navigator to install packages not included in the standard  Anaconda Navigator installer package. 
  1. open a terminal window from with Anaconda Navigator (see figure \ref{231311} below )
  2. issue the installation command specified in the package documentation. See below fexamples for conda and for pint
Note: When using Anaconda, the conda installer is preferable to the pip installer , if available (as conda will usually also install any additional needed extensions)
