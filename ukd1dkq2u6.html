Calculations with uncertainties

handling uncertainties in Pint

For  simple first-order ('linear') error propagation involving quantities with units,  you can use Pint's handy  plus_minus()  operator. It allows absolute and relative uncertainties: 
import numpy as np
from pint import UnitRegistry
unit = UnitRegistry()
width = (10. * unit.centimeter).plus_minus(.1, relative=True) # 10 percent uncertainty
length = (20. * unit.centimeter).plus_minus(2.) # 2 cm uncertainty
area = length * width
print(width)
print(length)
print('{:~P}'.format(area))

(20.0 +/- 2.0) centimeter
(200 ± 28) cm²
