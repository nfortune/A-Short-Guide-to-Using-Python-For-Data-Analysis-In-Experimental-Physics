saving data to a CSV file

 
We now provide an example of how to use the savetxt command from Numerical Python (numpy) to save data in csv format. There are admittedly many other ways to save files in spreadsheet format but this is the simplest method we've found so far using numpy. For more advanced data handling of spreadsheet data files and/o large data sets, you may wish to consider the popular Python Data Analysis Library package called pandas or the Hierarchical Data Format (HDF5)  Python interface package h5py .
