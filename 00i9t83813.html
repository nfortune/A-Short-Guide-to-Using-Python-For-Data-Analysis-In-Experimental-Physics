What these commands do:
%matplotlib inline is an interactive Python  (iPython) 'magic command' used when running Python within a Jupyter notebook. It allows the display of data plots within the notebook.  Include it in your code and it will magically do what you need it to do! Just be sure to follow the sequence shown above when using pyplot in Jupyter notebooks within Authorea by issuing the  %matplotlib inline command before  issuing the from matplotlib import pyplot command. 
The command plt.figure() signifies the beginning of the plotting instructions specific to that figure.  
The command plt.errorbar(angle, V_pd, xerr = None, yerr=V_pd_delta)  is the command that instructs matplotlib to generate a plot with error bars (as . The arguments linestyle, color, capsize, and capthick are optional commands to customize the appearance of the graph.  Linestyle = None means there are no connecting lines between points,   capsize =3 sets the width of the error bars to '3' (in typesetter points), and  capthick=1sets the thickness of the drawn bars to '1' (again in typesetter points). Note that if  the  capsize and capthick commands are omitted,  matplotlib will draw  lines indicating the given uncertainty but will omit the bars!
