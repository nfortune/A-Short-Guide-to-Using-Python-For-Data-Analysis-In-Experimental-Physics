Note:  %matplotlib inline is an iPython 'magic command' used when running Python within a Jupyter notebook to allow the display of data plots within the notebook.  Just include it in your code and it will magically do what you need it to do. To use pyplot in Jupyter notebooks within Authorea, be sure to issue this command BEFORE importing pyplot! Technically,  there are two steps t allows the interactive use of matplotlib
