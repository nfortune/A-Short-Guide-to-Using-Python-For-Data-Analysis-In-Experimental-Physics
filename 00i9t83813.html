What this does:
%matplotlib inline is an interactive Python  (iPython) 'magic command' used when running Python within a Jupyter notebook. It allows the display of data plots within the notebook.  Include it in your code and it will magically do what you need it to do! Just be sure to follow the sequence shown above when using pyplot in Jupyter notebooks within Authorea by issuing the  %matplotlib inline command before  issuing the from matplotlib import pyplot command.
