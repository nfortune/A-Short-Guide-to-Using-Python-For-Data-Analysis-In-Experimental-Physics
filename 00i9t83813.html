Note:  %matplotlib inline is an iPython 'magic command' used when running Python within a Jupyter notebook to allow the display of data plots within the notebook.  Include it in your code and it will magically do what you need it to do! Just be sure to follow the sequence shown above when using pyplot in Jupyter notebooks within Authoreaand issue the  %matplotlib inline command before the  import pyplot command
