An example of data smoothing using the Savitzky-Golay low pass filter is shown in Fig \ref{987576}.
Significantly Increasing the window width increases the degree of smoothing, causing small features in the original 'clean signal' to disappear; significantly decreasing the window width decreasing the degree of smoothing, causing artificial features (arising from the noise) to emerge. Increasing the order of the polynomial decreases the noise reduction but is needed for if there is significant data curvature over the width of the smoothing window.
