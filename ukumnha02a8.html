We call this the traditional method because this is what you will find in the examples included in the official numpy user guide and quickstart tutorials.  If you use this method, take note that you will  have to specify that you are using a function  or module from the numpy package each time you use that function. To do so, precede the name of the function with np.  This is why the loadtxt command in this method is written np.loadtxt instead of loadtxt.  In examples, the use of the  np.prefix helps identify when the numpy package is being used but in your own code it can be a nuisance to remember! The choice of methods is yours.
In both cases, we used the loadtxt command from Numerical Python (numpy) to load the data from a .csv file. loadtxt can also load other types of text files: see the numpy.loadtxt manual page for details. 
Note for Python geeks: if you are reading this, you probably know that if you use the direct method to type from numpy import * then  you can import all numpy commands at once instead of one at a time  .This import * approach is convenient but take two considerations in mind (1) you are importing code you don't need (2) you will overwrite an functions with the same name as a numpy function with the numpy version.
