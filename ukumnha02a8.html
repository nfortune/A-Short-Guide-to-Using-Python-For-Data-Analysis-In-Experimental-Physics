We call this the traditional method because this is what you will find in the examples included in the official numpy user guide and quickstart tutorials.  The key to this method is to be sure to specify that you are using a function  or module from the numpy package each time you use that function by preceding it either with numpy or the abbreviation of your choice (such as np).  This is why the loadtxt command in this method is written np.loadtxt instead of loadtxt.  
Finally, if your data isn't in CSV format, note that loadtxt can load many other types of text files as well. See the numpy.loadtxt manual page for details. 
 

saving data to a CSV file

 
We now show a relatively simple method of using the savetxt command from Numerical Python (numpy) to save the (angle, V_pd) data in csv format. It is almost as easy as the loadtxt command. There are many ways to do this but this is the simplest method we've found so far using numpy. For more advanced data handling of spreadsheet data files and large data sets, you may wish to consider the popular Python Data Analysis Library package called pandas or the Hierarchical Data Format (HDF5)  Python interface package h5py .
