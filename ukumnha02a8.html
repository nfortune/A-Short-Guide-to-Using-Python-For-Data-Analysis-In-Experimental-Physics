We now use the savetxt command from Numerical Python (numpy) to save our (angle, V_pd, V_pd_delta) data in csv format. 
Note: if we had typed savetxt( output_filename, (angle, V_pd, V_pd_delta, would save the data as three rows of data with each variable corresponding to a different row instead of two a different column — which is convenient for work within python but is not what we want in this particular case — we first did the following trick: we used the 2D array transpose command  to create a 2D matrix of our data, then transposed the matrix to flip the rows and columns.
