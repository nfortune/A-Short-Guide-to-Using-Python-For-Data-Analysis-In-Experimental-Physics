We call this the traditional method because this is what you will find in the examples included in the official numpy user guide and quickstart tutorials.  If you use this method, take note that you will  have to specify that you are using a function  or module from the numpy package each time you use that function by preceding it either with numpy or the abbreviation of your choice (such as np).  This is why the loadtxt command in this method is written np.loadtxt instead of loadtxt.  
If using the abbreviated prefix  np instead of numpy doesn't make you swoon, you don't have to use it, but its usage is ubiquitous. To skip it, type import numpy instead of import numpy as np and precede the loadtxt command with numpy., as in numpy.loadtxt. Note that the abbreviation can be anything you want. If you preferred whimsy over clarity, you could even type import numpy as turtles.  Don't! 
Finally, if your danote that loadtxt can load many other types of text files as well. See the numpy.loadtxt manual page for details.
